📘 Matemática – Equações do 1º e 2º Grau

✅ Equação do 1º Grau

Forma geral:

ax + b = 0

Solução: Isola-se o x:

x = -b / a

✅ Equação do 2º Grau

Forma geral:

ax² + bx + c = 0

Fórmula de Bhaskara:

x = (-b ± √(b² - 4ac)) / (2a)

Delta (Δ):

Δ = b² - 4ac
  • Δ > 0: 2 raízes reais diferentes
  • Δ = 0: 1 raiz real (repetida)
  • Δ < 0: raízes complexas (não reais)

✅ Soma e Produto das Raízes

Soma:

x₁ + x₂ = -b / a

Produto:

x₁ ⋅ x₂ = c / a

🧠 Problemas Contextualizados

Usar equações pra resolver:

  • Situações envolvendo tempo de percurso, área, velocidade
  • Questões com contexto militar (ex: lançamentos, rotas, distâncias percorridas)
